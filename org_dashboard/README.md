# 리얼에셋부문 조직 대시보드

## 개요

엑셀 조직표 `260325_리얼에셋부문_조직개편_인력배치_v1.4_RA.xlsx`를 웹 대시보드 형태로 재구성한 정적 산출물입니다.

구성은 다음과 같습니다.

- `build_dashboard_data.py`: 엑셀을 읽어 대시보드용 데이터 파일 생성
- `org-data.json`: 정규화된 원시 데이터
- `org-data.js`: 브라우저에서 바로 읽는 데이터 파일
- `index.html`: 메인 대시보드
- `styles.css`: 화면 스타일
- `app.js`: 필터, 검색, 조직 렌더링 로직

## 사용 방법

1. 데이터 새로고침

```powershell
cd "D:\Project\00. 2025 RA 기획추진\03. 부문 내 업무\00. RA_조직 관련\26년 조직개편(안)\final version\org_dashboard"
python .\build_dashboard_data.py
```

2. 대시보드 열기

- `index.html`을 브라우저에서 바로 열어도 됩니다.
- 필요하면 아래처럼 간단한 로컬 서버로 띄울 수 있습니다.

```powershell
python -m http.server 8787
```

그 후 브라우저에서 `http://localhost:8787/` 접속

## 현재 반영 로직

- 계층 구조: `섹션 > 그룹 > 파트 > 세부조직`
- 직급 구조: `그룹장 / 파트장·센터장 / 담당 / 시니어매니저 / 매니저`
- 태그 추출:
  - `(겸)` 포함 시 `겸직`
  - `(대)` 포함 시 `대행`
  - `외부영입` 표기 시 `외부영입`

## 확장 아이디어

- 특정 인물 클릭 시 겸직 관계 네트워크 표시
- 조직별 span of control 비교
- 원본 엑셀 교체 시 자동 최신본 탐색
- 조직개편안 버전 간 diff 비교
